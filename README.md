# QueMaster

A comprehensive question management and learning platform that helps students and learners master programming concepts across multiple languages.

## 📋 Overview

QueMaster is an intelligent question repository and learning management system designed to help students prepare for coding interviews and improve their programming skills. It provides curated questions, syllabi, and resources for multiple programming languages.

## 🎯 Features

- **Multi-Language Support**: C, C++, Java, and Python
- **Comprehensive Syllabi**: Structured learning paths with detailed syllabi for each language
- **Question Management**: Organized question bank for practice and revision
- **Web-Based Interface**: User-friendly web application built with Flask and modern web technologies
- **Interactive Learning**: Dynamic question generation and management system

## 📚 Supported Languages

- **C** - Learn systems programming and fundamentals
- **C++** - Master object-oriented programming
- **Java** - Explore enterprise-level programming
- **Python** - Get started with high-level programming

Each language comes with:
- Detailed syllabus PDF
- Organized question repository
- Learning progression guides

## 🛠️ Technology Stack

- **Backend**: Python with Flask
- **Frontend**: JavaScript, HTML, CSS
- **Database**: Integrated question management system
